# Chapter 4 Practice Set - Lists & Tuples

# Q1: Store 7 fruits entered by the user in a list
# Input:
fruits = ["Apple", "Banana", "Mango", "Orange", "Grapes", "Pineapple", "Guava"]
print(fruits)

# Output: ['Apple', 'Banana', 'Mango', 'Orange', 'Grapes', 'Pineapple', 'Guava']

---

# Q2: Accept marks of 6 students and display them in sorted manner
# Input:
marks = [78, 45, 90, 67, 88, 56]
marks.sort()
print(marks)

# Output: [45, 56, 67, 78, 88, 90]

---

# Q3: Demonstrate that a tuple cannot be changed
# Input:
t = (1, 2, 3)
print(t)

# Output: (1, 2, 3)

---

# Q4: Sum a list with 4 numbers
# Input:
numbers = [10, 20, 30, 40]
print(sum(numbers))

# Output: 100

# Q5: Count number of zeros in the following tuple
# Input:
a = (7, 0, 8, 0, 0, 9)
