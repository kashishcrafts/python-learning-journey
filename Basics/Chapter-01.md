# 📘 Chapter 1: Modules, Comments & pip

## 🔹 What is Programming?
Programming is the process of giving instructions to a computer to perform tasks.

---

## 🔹 What is Python?
Python is a simple and easy-to-read programming language.
It is beginner-friendly and used in web development, data science, AI, and automation.

---

## 🔹 Features of Python
- Easy to understand  
- High-level language  
- Free and open source  
- Portable (Windows, Mac, Linux)  
- Fun to work with

## Modules

A module is a file containing reusable Python code.

Types of Modules:
Built-in modules → os, random
External modules → flask, pyjokes

---

##🔹 pip

pip is the package manager for Python.

Example:
pip install flask

---

##🔹 Using Python as a Calculator

Python can be used as a calculator in REPL mode.

Example:

2 + 3
5 * 6
10 / 2

---

##🔹 Comments

Comments are used to explain code. Python ignores them during execution.

Single-line Comment:
# This is a comment

Multi-line Comment:
"""
This is a multi-line comment
"""
